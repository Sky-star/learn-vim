## 第三十六天

### chrome - devTools 高频操作

- 打开 devTools:
  - 开打上次的位置: `option + command + i`
  - 打开控制台列表: `option + command + j`
  - 打开元素列表: `option + command + c` 或 `shift + command + c`
- 控制台页面搜索文件: `command + p`
- 控制台页面搜索命令: `command + shift + p`
- 聚焦在控制台: ctrl + `
- 清空控制台: `command + k`
- 切换面板: `command + [ / ]`
- 在 sources panel 中 显示/隐藏底部面板: `ESC`
