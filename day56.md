## 第五十六天

### Obsidian - 内容编辑

- 切换阅读和编辑模式: cmd + e
- 加粗: cmd + b
- 斜体: cmd + i
- 添加链接: cmd + k
- 添加无需列表： - / cmd + -
- 代码块: `` / ```
- 注释: cmd + /
- Todo: cmd + enter
- 移动行: alt + 上/下
- 编辑当前文件标题: f2
- 代码折叠:

```
exmap foldAll obcommand editor:fold-all
nmap zM :foldAll
exmap unfoldAll obcommand editor:unfold-all
nmap zR :unfoldAll
exmap toggleFold obcommand editor:toggle-fold
nmap za :toggleFold
```
