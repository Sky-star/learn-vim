## 第四十七天

### Alfred

需要先将 mac 中的 spotlight 关闭，才可设置呼出快捷键为 alfred

- 激活: cmd + space
- 关闭: esc

- actions :

  - 进入: →
  - 退出: esc

- 进入到文件内: cmd + ↑/↓

  - open: return

- file search:

  - open: space / '
  - find: 可以用 cmd + return 代替
  - in
  - tags

- 基于 web search: ctrl + return

- 多文件操作:

  - 选择/取消: alt + ←
  - 从上到下取消: alt + ↓
  - 从下到上选择: alt + ←
  - action: alt + →

- bookmarks 搜索: bm

- web search:
  - ghs: github.com/search
  - bls: search.bilibili.com/all
  - npms: npmjs.com/search
