
## 第六天

### 文本对象的学习

```
文本对象是为了方便能够快速选择想要的范围而存在的,个人就是理解为范围
```

### 语法

```
操作符 + a/i + 对象(范围)

i: 选择对象范围内
a: 选择对象范围外
```

#### 对象

```
w： 一个词
b或(/)：一对()
[/]: 一对[]
{/} 或 B: 一对{}
</>: 一对<>
t： 标签内的内容，类似html，xml
s： 一个句子（不常用）
p： 一个段落（不常用）

函数参数的选择：
操作符+ ia 不包含分隔符的一个参数
操作符+ aa 包含分隔符的一个参数

```

### 技巧
```
daa： 删除一个参数，会删除一个分隔符
cia: 修改一个参数

dae: 删除当前文本所有内容
die： 删除当前文本所有内容，但是不会包含前后的空格
```
