## 第七天

### 在单文件中移动的更快

```
ctrl + f: 向下滚动一屏
ctrl + b: 向上滚动一屏
ctrl + d: 向下滚动半屏
ctrl + u: 向上滚动半屏
ctrl + e: 向下滚动一行
ctrl + y: 向上滚动一行
zz: 将当前行移动到屏幕中央
zt: 将当前行移动到屏幕顶部
zb: 将当前行移动到屏幕底部
gg: 移动到文件头部
G:  移动到文件尾部
line number + gg: 跳转到指定行数
line number + G : 跳转到指定行数
```
### 自定义配置

```
shift + k: 向上5行
shift + j: 向下5行
```
